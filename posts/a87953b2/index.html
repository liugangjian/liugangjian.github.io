Linear List

Linear List: A linear structure that consists of ordered sequences of data elements of the same type

  • The number of elements in the list is called the length of the linear list.
  • When the linear list has no elements, it is called an empty list.
  • The starting position of the list is called the header, and the ending position of the list is called the tail.
